# **Custom Advanced Calculator**

Live Demo https://alfareza26.github.io/ModernCalc/

##  **Deskripsi Proyek**

Kalkulator ini adalah proyek web interaktif yang dibangun menggunakan **HTML, CSS, dan JavaScript dasar**. Desainnya terinspirasi dari kalkulator modern dengan fokus pada **estetika, dan responsifitas**.

Beberapa fitur penting mencakup:

* Mode tampilan bertema (Default, Terang, Gelap)
* Tampilan display dua-layer seperti kalkulator profesional
* Dukungan input dari keyboard
* Highlight tombol ketika ditekan
* Hasil muncul sebagai layer terpisah

---

##  **Fitur Utama**

###  **1. Dual Display **

* **Main Display:** angka input utama (besar).
* **Result Display:** hasil perhitungan muncul di atas dengan ukuran lebih kecil.

###  **2. Tema Tampilan**

* Mendukung **tema Default, Light, dan Dark**
* Tema Light membalik kontras warna elemen UI (border, tombol, panel, dll)

###  **3. Highlight Tombol**

* Saat tombol ditekan melalui **mouse atau keyboard**, tombol di UI menyorot untuk memberi feedback visual.

###  **4. Tampilan Stabil**

##  **Metodologi (Design Thinking)**

Proyek ini menggunakan pendekatan **5 tahap Design Thinking**:

### 1. **Empathize**

Menganalisis kebutuhan pengguna kalkulator modern: tampilan bersih, respons cepat, input keyboard, display rapi.

### 2. **Define**

Permasalahan utama:

* Tampilan kalkulator HTML umumnya “kaku”
* Tidak ada pemisah ribuan
* Display terlalu fleksibel dan tidak realistis

### 3. **Ideate**

Merancang:

* Display dua layer
* Tema gelap/terang
* Efek highlight tombol
* Pemformatan angka otomatis

### 4. **Prototype**

Membangun versi awal dengan HTML + CSS + JS sederhana.

### 5. **Test**

Menyesuaikan berdasarkan:

* Input keyboard
* Konsistensi warna antar tema
* Stabilitas display
* Format angka ribuan/desimal
